Pour installer le cloud :
	- s'assurer que le serveur web ait les droits d'accès de modification et d'éxécution sur les fichiers, le dossier dans 
lequel est installé le cloud ainsi que celui où seront stockés les fichiers.
	- ouvrir avec le navigateur le fichier installation.php
	- remplir tous les champs
	- cliquer sur Installer
	- supprimer le fichier installation.php
